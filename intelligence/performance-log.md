# Performance Log — Suivi des publications

> Après chaque publication, noter les métriques pour alimenter l'intelligence de l'agent.
> Ces données permettent d'ajuster la stratégie éditoriale.

---

## Comment remplir

Après 48h-72h (quand les metrics se stabilisent), copier ce template :

```
## [DATE] — [TITRE/HOOK DU POST]
- **Pilier :** [IA & Transfo / Stratégie / Business / Neurosciences / Innovation / Coulisses]
- **Format :** [Post / Carrousel / Article / Newsletter / Vidéo]
- **Impressions :** 
- **Likes :** 
- **Commentaires :** 
- **Partages :** 
- **Taux d'engagement :** (likes + commentaires + partages) / impressions × 100
- **Nouveaux abonnés :** 
- **Observation :** [Ce qui a fonctionné / pas fonctionné selon toi]
```
